# Markdown Explorer v0.3.3

Small polish on top of v0.3.2.

## Fixed

- **Code blocks were double-spaced.** Inside a fenced code block, the renderer applied `paragraphSpacing = 8` to the whole highlighted attributed string — and since every newline ends a paragraph, that 8pt gap was being inserted between *every line of code*. Now the inner paragraph style has zero `paragraphSpacing`/`paragraphSpacingBefore`/`lineSpacing`, and the spacing above/below the block lives on dedicated padding lines.
- **In-progress sidebar had visible blank space between folders.** Compounded from `LazyVStack(spacing: 6)` outside + `VStack(spacing: 2)` inside each row + per-row vertical padding. Tightened to `LazyVStack(spacing: 0)` + inner `VStack(spacing: 0)` + a single top padding per row, so folder groups sit flush.

## Install

1. **Download** `MarkdownExplorer-v0.3.3.dmg` below.
2. **Open** the DMG, drag the app to **Applications**.
3. **First launch**: ad-hoc signed, so macOS will warn. Right-click → **Open** → **Open**, or **System Settings → Privacy & Security → Open Anyway** after the first refused launch.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon (arm64)
