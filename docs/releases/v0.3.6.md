# Markdown Explorer v0.3.6

## Fixed

- **Visible padding between folder groups in the In progress view is back to tight.** A leftover `.padding(.top, 6)` on each `WIPFolderRow` from the v0.3.3 reshuffle was adding 6pt above every folder. Removed; header now uses `.padding(.top, 4) .padding(.bottom, 2)` for a clear separation without ballooning. File rows shrunk from `.padding(.vertical, 2)` to `.padding(.vertical, 1)` so files inside a folder sit closer together.

Net result: 2-3pt between rows inside a folder, ~5pt above each folder header (enough visual separation for groups), no large blank bands.

## Install

1. **Download** `MarkdownExplorer-v0.3.6.dmg` below.
2. **Open** the DMG, drag the app to **Applications**.
3. **First launch**: ad-hoc signed, so macOS will warn. Right-click → **Open** → **Open**, or **System Settings → Privacy & Security → Open Anyway** after the first refused launch.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon (arm64)
