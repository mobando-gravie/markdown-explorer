# Markdown Explorer v0.3.2

## Fixed

- **Tables rendered as a single row.** A 2-column Field/Value markdown table was being laid out as one row with N+1 columns. Root cause: `NSTextTableBlock` was constructed with `startingRow: -1` / `startingColumn: -1`, which doesn't auto-assign — it puts every cell at row 0 with auto-incrementing columns. Now tracks `rowIndex` and `colIndex` explicitly and passes them into each block.

## Polished

- **Bigger headings**: h1 32pt, h2 24pt, h3 19pt (was 26 / 22 / 18) for the prominent GitHub-style hierarchy. h1 / h2 keep the bottom border. Adjusted paragraph spacing before headings.
- **Inline code background** is more visible in both light and dark mode. The run is now padded with hair spaces so the background isn't flush against surrounding glyphs.
- **List markers** ("• " for unordered, "1. " for ordered, "☐ / ☑" for task lists) have a wider hanging indent so the marker stays clearly to the left of the content.

## Install

1. **Download** `MarkdownExplorer-v0.3.2.dmg` below.
2. **Open** the DMG, drag the app to **Applications**.
3. **First launch**: macOS will refuse with the "cannot check for malicious software" warning (ad-hoc signed). Right-click → **Open** → **Open**, or **System Settings → Privacy & Security → Open Anyway** after the first refused launch.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon (arm64)
