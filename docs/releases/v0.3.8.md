# Markdown Explorer v0.3.8

## Changed

- **In progress refresh now sleeps when the app is inactive.** The 60-second background tick that re-scans projects in the In progress sidebar used to keep running even when Markdown Explorer wasn't the active app — burning IO and power for results no one could see. The timer now pauses on Cmd-Tab away (and on minimize / hide) and resumes only when you switch back. The moment the app becomes active again, an immediate scan runs before the next regular tick — so the first thing you see is fresh data, not 60 seconds of stale list. If the app sat idle long enough to cross the 1-hour bootstrap threshold, that escalation happens on the same focus-regain pass via the existing `tick()` path. The cached folder list stays on screen while paused, so coming back doesn't flash an empty sidebar.

## Install

1. **Download** `MarkdownExplorer-v0.3.8.dmg` below.
2. **Open** the DMG, drag the app to **Applications**.
3. **First launch**: ad-hoc signed, so macOS will warn. Right-click → **Open** → **Open**, or **System Settings → Privacy & Security → Open Anyway** after the first refused launch.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon (arm64)
