# Markdown Explorer v0.3.4

## New

- **Copy Path** added to the sidebar context menus alongside **Reveal in Finder**. Right-click any file row (or any folder header in the In progress view) → **Copy Path** puts the full filesystem path on the clipboard.

## Install

1. **Download** `MarkdownExplorer-v0.3.4.dmg` below.
2. **Open** the DMG, drag the app to **Applications**.
3. **First launch**: ad-hoc signed, so macOS will warn. Right-click → **Open** → **Open**, or **System Settings → Privacy & Security → Open Anyway** after the first refused launch.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon (arm64)
